# Face Recognition System

This project is a simple face recognition system developed in C++. It focuses on detecting and recognizing faces for basic data management.

## Features
- Face detection from images
- Basic face recognition
- Stores and processes user data

## Technologies
- C++
- OpenCV (if you used it)

## Project Structure
- main.cpp – main program logic
- train.cpp – training the model
- test.cpp – testing recognition
- faces/ – dataset for training

## How to Run
1. Compile the source files
2. Run the program
3. Follow the instructions in the terminal

## Notes
This project was built to explore pattern recognition and computer vision concepts.
